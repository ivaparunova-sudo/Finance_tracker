# Finance_tracker
Програмата представлява система за проследяване и анализ на личните финанси. Тя позволява въвеждане и обработка на месечни приходи и разходи, генериране на отчети, анализи и прогнози за бъдещото финансово състояние. Чрез нея потребителят получава ясна и обобщена информация за своя бюджет, спестявания и финансови навици за период до една година.
